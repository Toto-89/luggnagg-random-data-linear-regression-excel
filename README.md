# Luggnagg Random Data and Linear Regression in Excel

## Project overview
This Excel project was developed to simulate and analyze a realistic random dataset for a fictional population called **Luggnagg**.  
The workbook was designed to support statistical practice, data manipulation, correlation analysis, and linear regression using spreadsheet tools.

## Business context
In a data-driven environment, generating realistic synthetic data is useful for:
- testing statistical techniques
- training analysts on spreadsheet-based analysis
- simulating real-world scenarios without using sensitive data
- supporting evidence-based decision-making skills

This project focuses on creating and analyzing a sample of **250 individuals** whose ages follow a **normal distribution** based on user-defined parameters.

## Project objective
The main objective of the project is to build an Excel workbook that:
- generates random but meaningful data
- allows parameter-driven simulation
- supports statistical analysis of a selected sample
- explores correlation and non-correlation between variables
- applies linear regression to simulated data

## Workbook structure

### 1. Parameters
This sheet provides the input parameters used to generate the simulated population.

Included parameters:
- **Probability**
- **Mean**
- **StdDev**

Formatting features:
- header styled with **Comic Sans MS**, **12 pt**, **blue font**, and **double borders**
- value cells formatted with thin black borders

### 2. Data
This sheet contains the generated dataset for the population of 250 individuals.

Included columns:
- **Age**
- **Group**

Main features:
- ages generated from a normal distribution
- random group assignment with integer values from **1 to 4**
- formatted header consistent with project requirements

### 3. Sample
This sheet extracts a sub-sample from the generated data based on group membership.

Included columns:
- **Age**
- **Group**
- **Selected sample**

Main features:
- conditional logic used to select values from a specific group
- sample preparation for further statistical analysis

### 4. Statistical Insight
This sheet summarizes the selected sample through key statistical indicators.

Included metrics:
- **STDDEV**
- **EXPECTED VALUE**
- **COUNT**
- **CONFIDENCE RATE**
- **Estimation of p parameter**
- **Confidence Interval**

Purpose:
- evaluate the variability and expected behavior of the sample
- support interpretation of confidence and estimation measures

### 5. (Un)correlated Variables
This sheet explores the relationship between sample ages and additional random variables.

Included columns:
- **Sample data**
- **Number of cats**
- **Age of partner**

Main features:
- correlation testing between age and other simulated variables
- comparison between correlated and uncorrelated patterns
- interpretation of statistical relationships

### 6. Linear Regression
This sheet applies linear regression to the generated data.

Included columns:
- **Y (age)**
- **X (rank)**

Main features:
- linear regression analysis between age and rank
- scatterplot for visual interpretation
- estimation for participant **160**
- regression-based analytical output
